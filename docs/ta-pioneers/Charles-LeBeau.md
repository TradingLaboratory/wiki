# Charles LeBeau (годы жизни уточняются)
*Чарльз Лебо*

> «Результат каждой сделки зависит от выхода. Если мы вошли вовремя, а вышли плохо — сделка, скорее всего, окажется убыточной»  
> *«The outcome of every trade is dependent on the exit. If we enter in a timely fashion and then exit poorly, the trade is likely to be a loss»*

## Основной вклад

- Пионер компьютеризированного бэктестинга торговых систем: один из первых разработал методологию количественной оценки эффективности стратегий выхода из позиций.
- Создал концепцию **Chandelier Exit** — динамического трейлинг-стопа на основе волатильности (ATR), позволяющего «висеть» под максимумом тренда и защищать прибыль без преждевременного выхода.
- Сместил фокус трейдеров с поиска точек входа на систематизацию правил выхода, доказав, что качество выходов критичнее для прибыльности системы.
- В соавторстве с Дэвидом Лукасом написал фундаментальное руководство по построению и тестированию торговых систем, ставшее стандартом для поколения системных трейдеров.

## Индикаторы, используемые до сих пор

| Индикатор | Общепринятое название | Назначение | Базовый период |
|-----------|----------------------|------------|----------------|
| Chandelier Exit (Long) | Chandelier Exit / Выход по принципу люстры (лонг) | Трейлинг-стоп для длинных позиций: отслеживает максимум тренда с учётом волатильности | 22 / множитель 3 |
| Chandelier Exit (Short) | Chandelier Exit / Выход по принципу люстры (шорт) | Трейлинг-стоп для коротких позиций: отслеживает минимум тренда с учётом волатильности | 22 / множитель 3 |

## Предупреждение для программистов

При реализации Chandelier Exit критически важно:
- Для лонгов использовать **наивысший максимум** (highest high) за период, а не цену закрытия
- Для шортов использовать **наименьший минимум** (lowest low) за период
- Период расчёта ATR и период поиска экстремума могут отличаться (Лебо рекомендует 22 для обоих, но в практике часто используют 14 для ATR и 22 для экстремума)
- Индикатор **не генерирует сигналы входа** — только точки выхода/защиты прибыли

## Практическое применение в стратегиях

| Задача | Решение через индикаторы Лебо |
|--------|-------------------------------|
| Защита прибыли в тренде без преждевременного выхода | Chandelier Exit «висит» под максимумом, смещаясь только при росте цены, но не опускаясь при коррекциях |
| Адаптация стопа к изменению волатильности | Множитель ATR автоматически расширяет/сужает дистанцию стопа в зависимости от рыночной волатильности |
| Фиксация прибыли при развороте тренда | Пробой линии Chandelier Exit после длительного тренда сигнализирует об ослаблении импульса |
| Управление риском в волатильных активах | Для высоковолатильных инструментов увеличение множителя ATR (с 3 до 4–5) предотвращает ложные срабатывания |
| Комбинация с трендовыми фильтрами | Использование Chandelier Exit только при подтверждённом тренде (например, цена выше 200-дневной SMA) повышает эффективность |
| Выход из убыточных позиций | Применение укороченного периода (10 вместо 22) для быстрого выхода при неблагоприятном развитии |

## Философия

- **Выход важнее входа**: даже идеальный вход не спасёт сделку при ошибочном экзите; прибыльность системы определяется правилами выхода [[87]]
- **Волатильность как основа управления риском**: стоп-лосс должен масштабироваться с рыночной волатильностью, а не быть фиксированным в пунктах или процентах
- **Простота и воспроизводимость**: эффективные стратегии выхода должны быть механическими, без субъективной интерпретации, чтобы обеспечить статистическую достоверность бэктеста

## Кратко для разработчика торговой стратегии

```
# Chandelier Exit (лонг)
period = 22          # период поиска максимума
atr_period = 22      # период ATR (может отличаться)
multiplier = 3.0     # множитель волатильности

highest_high = max(high[-period:])
atr_value = ATR(atr_period)[-1]
chandelier_long = highest_high - atr_value * multiplier

# Chandelier Exit (шорт)
lowest_low = min(low[-period:])
chandelier_short = lowest_low + atr_value * multiplier

# Правила использования:
# 1. Для лонгов: выход при цене закрытия < chandelier_long
# 2. Для шортов: выход при цене закрытия > chandelier_short
# 3. Не использовать как сигнал входа — только для выхода/трейлинга
# 4. При высокой волатильности увеличить множитель до 4–5
# 5. Для скальпинга сократить период до 10–14
```
